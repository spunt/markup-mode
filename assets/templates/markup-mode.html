Markup Mode — live demo

Turn on review mode, then select text or click an element to leave a note. This page is also the copyable reference — grab the block between the markers below and paste it into any artifact before the closing body tag.

What it does

Markup Mode adds a frontend-only review layer to any page. A reviewer anchors a note to a specific run of text (a claim, a sentence) or to a specific element (a card, a chart), then compiles every note into a tagged Markdown bundle to hand back to an agent.

It writes nothing into the host's content and needs no backend: notes live in localStorage and leave as Markdown via Copy or Export.

Two anchor types

Text anchor

Drag-select any words. The selection snaps to word boundaries, is highlighted in place, and the exact quote is recorded.

Element anchor

Click any element; press the arrow keys to grow or shrink the pick. A numbered mark labels the target unambiguously.

Try it

Click Start Markup Mode in the dock (bottom-right) or press Cmd/Ctrl+Shift+K. Leave a couple of notes, click a mark to edit, then Compile and Copy. On a wide screen, use the dock-to-right button to make the panel a resizable column.
